# 💧 Liquid Glass Ripple Effect Button

A futuristic **Glassmorphism Button** with a realistic liquid ripple click animation.
Built using pure **HTML, CSS, and Vanilla JavaScript** — lightweight, modern, and beginner friendly.

---

## 🚀 How It Works

1. When the button is clicked, a new `<span>` element is created dynamically.
2. The ripple is positioned based on the mouse click coordinates.
3. CSS animation scales the ripple outward.
4. The ripple fades smoothly and removes itself automatically.

---

## ▶️ How to Use

1. Download or clone the repository.
2. Open the folder `ripple-effect`.
3. Open `index.html` in your browser.
4. Click the button and enjoy the liquid ripple effect.
